Support¶

The ObsPlus Github page is the place where ObsPlus development, support and discussion takes place. If you encounter and issue, discover a bug, or want to ask a question please first search the issues to see if your issue has already been addressed. If not, file a new issue by clicking on the green “New Issue” button and selecting the appropriate issue category.
